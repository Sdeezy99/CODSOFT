PROJECT TITLE: ATM Interface
PURPOSE OF PROJECT: This project mimics a ATM and a user can withdraw, deposit and check balance in their bank account.
VERSION or DATE: 10 Jan 2024
HOW TO START THIS PROJECT: Using BlueJ or any integrated development environment (IDE) compatible
AUTHORS: Mpho Madini
USER INSTRUCTIONS:

- User has to select options from 1 to 4 which include exiting, withdrawing , depositing and checking balance.
- After selecting options 1-3 you are given an opportunity to make another transaction
- User can exit the program by selecting option 4.
